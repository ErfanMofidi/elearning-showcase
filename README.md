# 🎓 E-Learning Showcase

A modern, responsive front-end website built to showcase an e-learning platform. This project demonstrates my ability to design and develop clean, interactive, and user-friendly interfaces using modern front-end technologies.

## 🌐 Live Demo

👉 https://erfanmofidi.github.io/elearning-showcase/

---

## 📖 About the Project

This project is a front-end implementation of an online learning platform. It focuses on creating an engaging user experience with responsive layouts, smooth animations, and reusable UI components.

The goal of this project was to practice building a professional-looking educational website while applying responsive design principles and modern CSS techniques.

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean UI
- Smooth scrolling and animations
- Course showcase section
- Instructor section
- Testimonials
- Pricing plans
- Contact section
- Interactive navigation
- Cross-browser compatible
